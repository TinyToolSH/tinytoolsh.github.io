**TinyToolSH**, or just tinytools, is an acronym for Tiny Tools written in SHELL script.
Tinytools is a collection of mostly bash scripts. Our projects are designed to improve your productivity in a simple way.

## Our Projects

### Journalling
* [tnotes](/tools/tnotes) - A tiny file-based note manager.
* [ttodo](/tools/ttodo) - A simple dmenu wrapper to create and access your todo list.
* [tbm](/tools/tbm) - A tiny tool to create url bookmarks in plain texts.
* [tpomodoro](/tools/tpomodoro) - A simple script to manage your time using the pomodoro technique.

### Menu-driven applications
* [tsearch](/tools/tsearch) - A simple CLI tool to create and launch search requests.
* [tlaunch](/tools/tlaunch) - A menu composer to launch commands.
* [tgit](/tools/tgit) - A command to list all the important repositories in your local system.
* [tmenu](/tools/tlaunch) -A tiny wrapper for menu utilities with support for dmenu, rofi, fzf and others.

### Utilitites
* [tyaml](/tools/tyaml) - A simple script to parse a yaml file.
* [tprogbar](/tools/tprogbar) - A simple commandline tool to help you make your our ASCII progress bar.
* [tmdpres](/tools/tmdpres) - A tiny markdown presentation generator.
* [tgoeswall](/tools/tgoeswall) - make Geostationary Operational Environmental Satellite images, your wallpaper.
